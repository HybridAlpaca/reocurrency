# reocurrency

ReoCurrency prototype app (nuts)

## instructions

These instructions assume you use `yarn` as your default package manager.  If you use NPM, consult the docs for commands.

### [1] building
```
yarn
yarn build
```

### [2] running
```
yarn app
```
